# تولیدکننده بریف محتوا با هوش مصنوعی (AI Content Brief Generator)

یک اپلیکیشن تک‌صفحه‌ای حرفه‌ای، مینیمال و مدرن که با لاراول ساخته شده و با قدرت هوش مصنوعی، بر اساس کلمات کلیدی، بریف‌های کامل محتوایی تولید می‌کند.

## 🚀 ویژگی‌ها

- **تحلیل مبتنی بر هوش مصنوعی**: تولید پلن کامل مقاله با استفاده از مدل‌های پیشرفته AI (gpt-oss-120b).
- **بهینه‌سازی سئو**: ارائه عنوان H1 پیشنهادی، Meta Description و ساختار کامل مقاله (H2/H3).
- **تحقیق کلمات کلیدی**: شامل کلمات کلیدی LSI و سوالات متداول پیشنهادی برای رتبه‌بندی بهتر.
- **پشتیبانی بین‌المللی**: پشتیبانی از چندین زبان و کشورهای هدف (فارسی، انگلیسی، عربی و غیره).
- **رابط کاربری مدرن**: طراحی تمیز، ریسپانسیو و Pixel-perfect با افکت‌های Glassmorphism و انیمیشن‌های نرم.
- **گزینه‌های خروجی**: امکان خروجی گرفتن به صورت PDF، کپی در حافظه یا چاپ مستقیم.

## 📋 پیش‌نیازها

- **PHP**: >= 8.2
- **Composer**: آخرین نسخه
- **وب‌سرور**: Apache، Nginx یا استفاده از سرور داخلی PHP (Artisan)
- **پایگاه داده**: اختیاری (در حال حاضر برای سادگی از درایور `file` برای سشن و کش استفاده شده است)

## 🛠️ مراحل نصب

برای راه‌اندازی پروژه به صورت محلی مراحل زیر را دنبال کنید:

۱. **کلون کردن مخزن**:
   ```bash
   git clone <repository-url>
   cd content-brief-generator
   ```

۲. **نصب وابستگی‌ها**:
   ```bash
   composer install
   ```

۳. **تنظیمات محیطی (Environment)**:
   فایل نمونه تنظیمات را کپی کرده و تنظیمات خود را اعمال کنید:
   ```bash
   cp .env.example .env
   ```

۴. **تولید کلید اپلیکیشن**:
   ```bash
   php artisan key:generate
   ```

۵. **اجرای برنامه**:
   سرور توسعه محلی را اجرا کنید:
   ```bash
   php artisan serve
   ```
   برنامه در آدرس `http://127.0.0.1:8000` در دسترس خواهد بود.

## 🤖 اتصال هوش مصنوعی دلخواه

شما می‌توانید به سادگی مدل هوش مصنوعی را تغییر دهید یا از سرویس‌دهنده اختصاصی خود استفاده کنید. این سیستم از ساختار استاندارد APIهای سازگار با OpenAI استفاده می‌کند. برای تنظیم AI خود:

۱. فایل `.env` را باز کنید.
۲. متغیرهای زیر را ویرایش کنید:
   - `AI_BASE_URL`: آدرس پایه سرویس‌دهنده شما (مثلاً `https://api.openai.com/v1`).
   - `AI_API_TOKEN`: توکن یا کلید امنیتی API شما.
   - `AI_MODEL`: نام مدلی که قصد استفاده از آن را دارید (مثلاً `gpt-4`).

برنامه انتظار دارد که هوش مصنوعی پاسخ‌ها را در قالب JSON ساختاریافته (طبق پرامپت سیستم) ارسال کند.

## 📖 نحوه استفاده

۱. **کلمه کلیدی اصلی** خود را وارد کنید.
۲. **زبان/کشور هدف** را انتخاب کنید.
۳. (اختیاری) **توضیحات** تکمیلی یا **URL رقبا** را وارد کنید.
۴. روی دکمه **"تولید بریف محتوا"** کلیک کنید و منتظر پردازش هوش مصنوعی بمانید.
۵. بریف خود را بررسی کرده و از دکمه‌های **کپی** یا **دانلود PDF** برای خروجی گرفتن استفاده کنید.
